---
name: Onboarding Generator
description: Generate onboarding documentation for a software repository.
---

When a user asks for onboarding documentation:

1. Search the repository structure.
2. Read the README.md.
3. Read the main application files.
4. Identify:
   - Project purpose
   - Entry point
   - Folder structure
   - Capabilities
   - Technologies
   - Installation steps
5. Generate a concise onboarding guide.
6. Save it as ONBOARDING.md using the write_file tool.
